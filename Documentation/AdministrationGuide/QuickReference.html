Quick Reference

The following table provides a quick reference of commands, file structure, agents, and available utilities for each supported operating system. In the table, $prefix defaults to /usr/local, but it can be changed by specifying --prefix=pathname in the autogen.sh script.

Binary Directories $prefix/bin, $prefix/sbin
Mailbox Directory $prefix/var/bongo/users/username
SCMS Directory $prefix/var/bongo/scms
Message Queue $prefix/var/bongo/spool
Template Directory $prefix/lib/modweb
SSL Certificate $prefix/var/bongo/dbf/osslcert.pem
SSL Key File $prefix/bongo/dbf/osslpriv.pem
WebAdmin SSL Certificate $prefix/sbin/osslcert.pem
WebAdmin SSL Key File $prefix/sbin/osslpriv.pem
Startup Commands /etc/init.d/bongo start
Stop Commands /etc/init.d/bongo stop
