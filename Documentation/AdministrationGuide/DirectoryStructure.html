The Directory Structure

The following table lists the Bongo directory structure. In the table, $prefix defaults to /usr/local. It can be changed by specifying --prefix=pathname in the autogen.sh script.

Binary Directories $prefix/bin, $prefix/sbin
Log Directory $prefix/var/log/messages
Mailbox Directory $prefix/var/bongo/users/username
SCMS Directory $prefix/var/bongo/scms
Message Queue $prefix/var/bongo/spool
Certificates $prefix/var/bongo/dbf
WebAdmin Certificates $prefix/sbin
Startup Commands /etc/init.d/bongo start
Stop Commands /etc/init.d/bongo stop
Template Directory $prefix/lib/modweb
