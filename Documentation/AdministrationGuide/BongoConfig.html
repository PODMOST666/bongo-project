Bongo-Config

bongo-config is a new Bongo utility that enables you to do several things:

  • User administration, up to and including:
    • Adding and removing users
    • Changing users passwords
  • Install base Bongo data (used during inital setup and reinstalls)
  • Generate cryptography information for Bongo
  • Check versioning information of Bongo online
  • Update timezone information on the local system
