Calendar Server in Bongo 1.0

Calendaring is a feature strongly desired for Bongo 1.0. Implementation options have yet to be explored, though. Spec:
  • CalDav or WCAL(?) implementation sufficient to support operations in:
    • Sunbird
    • Lightning
    • Evolution (?)
  • Able to view the same calendar data as in Dragonfly
To be thought about:
  • sharing of calendars
  • support for non-user resources with calendars.
