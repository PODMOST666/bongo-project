Architecture

Bongo is made up of several components:

  • Core libraries (msgapi, etc)
  • Core binaries/utilities (bongo-manager, bongo-config, ...)
  • Agents (bongosmtp, bongoimap, ...)
  • Hawkeye - Admin interface
  • Dragonfly
