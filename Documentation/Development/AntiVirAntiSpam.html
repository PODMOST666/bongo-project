Antivirus / Antispam requirements for Bongo 1.0

Items marked (?) are not considered hard requirements and could be dropped.

  • support use of RBLs
  • support use of greylisting
  • support external applications:
    • SpamAssassin
    • ClamAV
    • MailScanner (?)
  • some kind of spam/virus UI in Dragonfly front end (?) [Use cases: marking e-mails as spam, or scams, possibly some UI to train the software]
