Client Compatibility

As Bongo is predominantly a web based application, it would be nice to know what browser works without issues. A simple usability test was run, by connecting to a Bongo server from a variety of browsers on Linux and Windows.

Linux:

  • Opera 9 - Fine
  • FireFox 1.5.8 - Fine
  • FireFox 2 - Fine
  • Konqueror 3.5.5 - Works with issues. Login screen not very clear, login labels scrambled, colours not what they should be, log out doesn't work
  • Epiphany 2.16.1 - Fine
  • SeaMonkey 1.1 - Fine
  • Lynx 2.8.6 - Failed no JavaScript support
  • Links 2.1 - Failed no JavaScript support

Windows:

  • Opera 9 - Fine
  • FireFox 1.5.8 - Fine
  • FireFox 2 - Fine
  • Internet Explorer 6 - Fine
  • Internet Explorer 7 - Fine, but favicon doesn't display correctly
